Morphic Config Builder

Instructions

The Morphic Config Builder is a tool for creating config.json files for Morphic Enterprise sites.

This tool will generate a config.json file from scratch, or allow you to upload and edit an existing config.json file.

You can use different config.json files for different groups of computers at your site. Just create a config.json file for each group of computers and install it on all computers in that group.

To create and deploy a config.json file, follow these steps:

  1. Upload Current config.json (Optional - you can start with defaults below)
  2. Configure config.json Settings
  3. Download new config.json
  4. Install it on all computers you want to configure.

Group 3: Custom MorphicBar Buttons

Enterprises can add up to three extra buttons on the MorphicBar using the config file.

The three buttons can be the same for all computers at the organization, or different for different locations. For example, a school may want a different set of 3 buttons for computers in the classrooms than they do for the computers in the library, or labs or tutor center or in general areas.

This can be easily done but just creating different config files for the computers in the different locations.

Available Options

Below is a list of the standard 'extra' buttons from Morphic. In addition, at the bottom, are places where 3 custom buttons can be defined by the organization. You can choose any combination of 3 buttons from the custom and standard buttons.

Instructions
  • Fill in the numbers 1 2 and 3 before the buttons you want. You will see them appear in that location in the MorphicBar at the bottom of the page
  • For custom buttons, just fill in the blanks for that button. The appearance of the button and the help text for it will be shown for you as you create the buttons.
  • You do not have to add 3 buttons – but that is the max you can add at this time.
